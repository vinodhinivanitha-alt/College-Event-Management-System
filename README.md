# 🎓 College Event Management System

A web-based College Event Management System developed using **Python (Flask)** and **SQLite/MySQL**. It helps administrators manage college events and allows students to register, make payments, and provide feedback.

## ✨ Features
- Admin & Student Login/Register
- Event Creation & Management
- 40+ Cultural, Technical & Non-Technical Events
- AI-generated Event Descriptions & Instructions
- Organizer Name & Contact Details
- Event Registration
- Online & Spot Payment
- Feedback Module
- Reports Dashboard

## 🛠 Technologies Used
- Python
- Flask
- SQLAlchemy
- SQLite / MySQL
- HTML
- CSS

## 🚀 Run the Project

```bash
pip install -r requirements.txt
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
